Website Copy Recipe Website Animated Page
For this challenge, we had to copy a website from scratch and attempt to replicate it as well as possible. I learned a lot about vanilla CSS and HTML while attempting this because it requires you to understand how flexboxes work.
As a part of our learning experience, we were tasked with creating a multipage website about a topic of our choice. I chose to write about one of my favorite foods, Lumpia. During this exercise, we practiced linking images, videos and other forms of media onto a webpage. We also had the choice to use rows and columns to divide the content.
After we had a better understanding of CSS and HTML, we were then challenged to create a more dynamic website that included animations. Animations involve manipulating CSS properties so that images or text will move on screen as designed by the user. This is both a difficult but rewarding thing to do as it is very meticulous but also adds a lot of value to a website.
